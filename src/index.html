How to Use

1
Open iCloud

Click "Open iCloud" to open a window to iCloud's Data Recovery page.

2
Sign In & Open Restore Files

In the browser window, sign in with your Apple ID and complete 2FA. Then click "Restore Files" to open the file list.

3
Wait for Items to Load

Let the file list start loading. You don't need to wait for all items -- a few hundred is enough to start.

4
Select Items

Set your batch size (default 500) and click "Select" to automatically check that many unchecked items.

5
Restore / Confirm

Click "Restore / Confirm" to click the Restore button. After the restore completes, click it again to dismiss the confirmation dialog.

6
Repeat

Repeat steps 4-5 for the next batch. Keep going until all items are restored.

A
Auto Mode

Click "Auto Mode" to run hands-free overnight. Set the wait time (default 5 min) using the input next to the button. It will automatically select items, wait, click restore, wait, and repeat. A countdown bar shows time until the next action. Click "Stop Auto" to stop at any time.

Tip: If buttons aren't working, make sure items have loaded in the iCloud window first.
